## Purpose
Let parents quickly review daily baby activities.
The screen should feel calm, warm, and breathable.

## Visual Intent
- Soft
- Pastel
- Minimal but expressive
- Icons lead, text follows

## Key Components

### Custom Activity Segments
- NOT a TabBar
- Custom segmented control
- Large PNG icons
- Clear active state
- Touch-friendly spacing

### Logs
- Icon is the primary visual element
- Icon must be clearly visible at first glance
- Text is secondary
- No cramped rows
- No text overflow

### Summary Sections
- Compact
- Informational
- Icons smaller than log icons
- Must never break layout

## Do NOT
- Reintroduce TabBar
- Shrink log icons below visual clarity
- Change colors without request
- Apply global refactors
